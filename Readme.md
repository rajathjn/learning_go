# Learning Go

This repository contains my code and notes as I learn the Go programming language.

## Contents

### Basics
- **`basics/`**: Fundamental Go syntax and concepts
  - `hello.go` - Hello World program
  - `calc.go` - Basic calculations

### Data Structures
- **`arrays_slice_maps/`**: Working with arrays, slices, and maps
  - Array initialization and manipulation
  - Slice operations and append functionality
  - Map usage and operations

### Control Flow
- **`loops/`**: Loop constructs in Go
  - Traditional for loops
  - Range-based loops
  - While-style loops

### String Operations
- **`strings/`**: String manipulation and operations

### Object-Oriented Programming
- **`oops/`**: Interfaces and methods in Go
  - Shape interface implementation
  - `circle/` - Circle struct with Area and Scale methods
  - `rectangle/` - Rectangle struct with Area and Scale methods

### Concurrency
- **`concurrency/`**: Concurrent programming in Go
  - Goroutines with `go` keyword
  - Channels for communication
  - WaitGroups for synchronization
  - Select statements for coordination

## Requirements

- Go 1.25.5 or higher

## Basic Commands

- `go run <filename>.go` - Compile and run a Go program
- `go build <filename>.go` - Compile a Go program
- `go mod init <module_name>` - Initialize a new Go module
- `go mod tidy` - Add missing and remove unused modules

## Learning Resources

A principle of Go, I like: **"Don't communicate by sharing memory; share memory by communicating."**
